Meu Portfólio

Projeto 1

Desenvolvi uma calculadora simples para ajudar na gestão de despesas pessoais. Os usuários inserem valores de gastos mensais em diferentes categorias, como alimentação, transporte e entretenimento. A aplicação fornece um resumo claro e visual dos gastos, ajudando os usuários a manterem o controle de suas finanças.

Projeto 2

Desenvolvi uma lanterna de bolso utilizando Arduino, LEDs e uma bateria. A lanterna possui modos de iluminação intuitiva e é um projeto divertido para iniciantes em eletrônica. Utilizei programação em C para controlar os LEDs, proporcionando uma introdução prática à programação embarcada.
